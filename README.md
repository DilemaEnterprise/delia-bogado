# Delia Bogado — Landing (GitHub Pages)

## O que tem aqui
- `index.html` → a landing page
- `assets/` → imagens (placeholders)

## Como publicar no GitHub Pages
1) Crie um repositório (ex.: `delia-landing`)
2) Envie estes arquivos para a raiz do repositório
3) Vá em **Settings → Pages**
4) Em **Source**, selecione:
   - Branch: `main`
   - Folder: `/ (root)`
5) Salve e copie o link que o GitHub mostrar.

## Onde editar (rápido)
Abra o `index.html` e substitua:
- `[CIDADE]`
- `[WHATSAPP]`
- `[LINK_WHATSAPP]` (use o modelo abaixo)
- `[ENDEREÇO]`
- `[INSTAGRAM]` e `[INSTAGRAM_URL]`
